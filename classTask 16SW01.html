Define HTML 5

Learn the basics of Web design and style to give your Web sites a professional look and feel . The Microsoft team and experts from theWorld Wide Web Consortium (W3C) will guide you step-by-step in how to use the latest Web standards to create a site to be proud of. During the course, you will learn the basic building blocks of Web design and style – HTML5 and CSS – to give your site a professional look and feel. By the end of the course, you should understand all the fundamental elements – from headers and links to images and sidebars – and leave having built a basic framework for your own Web site.
